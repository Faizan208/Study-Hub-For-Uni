# **App Name**: LGU Study Hub

## Core Features:

- Firebase Authentication: Enable user sign-in using Firebase Authentication for secure access.
- Firestore User Profiles: Store basic user information (name, email) in Firestore.
- Resource Browsing: Allow users to browse and search available study resources.
- Resource Upload: Enable authenticated users to upload new study resources.
- Responsive Layout: Ensure the website is fully responsive across desktop, tablet, and mobile devices.
- Floating UI Elements: Add floating animated UI elements and icons (assignments, papers, graduation cap) to enhance visual appeal using CSS keyframes and Tailwind animations.

## Style Guidelines:

- Primary color: Deep red (#C30010) for headings and key elements.
- Accent color: Bright orange (#FF4000) for secondary highlights.
- Background color: Very light pinkish-white radial gradient (#FAE1E3) fading to white (#FFFFFF).
- Font: 'Inter' (sans-serif) for all text, providing a modern and clean look. Note: currently only Google Fonts are supported.
- Use simple, clear icons related to study materials (book, lightbulb, graduation cap).
- Pixel-perfect layout matching the reference image, focusing on a clean and intuitive user interface.
- Subtle animations for floating elements, button hover effects (scale and shadow), and navbar underline transitions using Tailwind's animate utilities and custom keyframes.
